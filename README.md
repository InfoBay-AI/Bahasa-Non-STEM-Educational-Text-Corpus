# Bahasa-Non-STEM-Educational-Text-Corpus

**Dataset Description:**

This dataset is a **large-scale collection of Bahasa Non-STEM Textbook data**, designed to support the development of advanced NLP systems and AI models for educational understanding, comprehension, and language learning in Bahasa.

Additionally, this dataset can be used in pipelines for **Supervised Fine-Tuning (SFT) and Reinforcement Learning with Human Feedback (RLHF) workflows**, improving model performance in text comprehension, summarization, and contextual reasoning tasks.

**Key Use Cases**

    -Text-based Question Answering (Descriptive & MCQ)
    -Named Entity Recognition (NER) in non-STEM content
    -Text summarization and comprehension
    -Automated tutoring and educational assistants
    -Knowledge retrieval systems for humanities subjects
    -Model evaluation and benchmarking

**Dataset Specification**

    -Modality: Bahasa
    -Type: Educational / Non-STEM
    -Data Source: Curated academic textbooks and educational material
    -Data Nature: Real-world and curated data
    -Books: 3,396

**Value of This Dataset**

    -Enables learning of non-STEM subjects in Bahasa
    -Improves comprehension and contextual reasoning in AI models
    -Supports multilingual and domain-specific NLP systems
    -Helps build AI-powered educational platforms
    -Enhances accuracy and reliability of LLMs in humanities domains
